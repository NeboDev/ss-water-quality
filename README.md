# ss-water-quality
Sistema de Gestión para el ODS 6: Agua y Saneamiento

Este proyecto forma parte de mi **Servicio Social** en la **Facultad de Ciencias de la Computación (BUAP)**. El objetivo es proporcionar herramientas tecnológicas que contribuyan a las metas del Objetivo de Desarrollo Sostenible 6 de la ONU.

## Propósito del Proyecto
Desarrollar un sistema de monitoreo/gestión que permita:
* Monitorear la calidad del agua en tiempo real.
* Optimizar la distribución de recursos hídricos.
* Generar reportes para el Insistuto de Ciencias de la BUAP.
